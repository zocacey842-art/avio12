# Aviator Crash Game

## Overview
A simple browser-based Aviator crash game built with HTML, CSS, and JavaScript. The game simulates a multiplier that increases over time, and players must cash out before the plane "flies away" (crashes).

## Project Structure
- `index.html` - Main game file containing all HTML, CSS, and JavaScript
- `server.py` - Simple Python HTTP server to serve the static files

## Running the Project
The project runs on a Python HTTP server bound to port 5000. Start with:
```bash
python server.py
```

## Game Features
- Betting system with configurable amount
- Real-time multiplier animation
- Airplane animation that moves as multiplier increases
- Cash out functionality before crash
- Provably fair crash point generation with 3% house edge

## Technical Details
- Language: Amharic (Ethiopian)
- Stack: Vanilla HTML/CSS/JavaScript
- Server: Python http.server
- Port: 5000
